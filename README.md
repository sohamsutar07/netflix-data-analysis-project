🎬 Netflix Data Analysis Project | Excel, SQL, Python, Power BI

🛠 Tools Used

• SQL – Business analysis
• Python – EDA & visualization
• Power BI – Dashboard

📊 Key Work

• Performed SQL analysis on content distribution
• Created Python charts for trends and insights
• Built interactive Power BI dashboard

• Content growth increased in recent years
